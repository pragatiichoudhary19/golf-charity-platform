# ⛳ Fairway Gives — Golf Charity Platform

A production-ready full-stack web application where golf players 
can track their scores and support their favourite charities.

## 🌐 Live Demo
[View Live App](https://golfcharity-platform.vercel.app)

## ✨ Features
- 🔐 Email/password authentication
- ⛳ Golf score tracking (max 5 scores, rolling window)
- 🎲 Weekly draw system with score matching
- 💚 Charity selection and donation tracking
- 👤 User profile with avatar
- 🛠️ Admin panel to run draws and view users

## 🛠️ Tech Stack
- **Frontend:** React 18 + Vite + Tailwind CSS
- **Backend:** Supabase (Auth + PostgreSQL)
- **Deployment:** Vercel

## 🚀 Getting Started
1. Clone the repo
2. Run `npm install`
3. Add `.env` with Supabase credentials
4. Run `npm run dev`
